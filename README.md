### How to run

```
# Without docker
pip install -r requirements.txt
NEXUS_TOKEN=<nexus api token> BOT_TOKEN=<discord bot token> python -m vdbot

# With docker
NEXUS_TOKEN=<nexus api token> BOT_TOKEN=<discord bot token> docker-compose up --build
```
